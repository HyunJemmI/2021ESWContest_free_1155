Software for Hard of Hearing

odas - ODAS(Open EmbeddeD Audition System) 폴더 //

__pycache__ - 캐시 파일 //

Classification.py - 소리 분류를 위한 모듈 //

HappyNewEar.sh - 프로그램의 실행 파일 //

log.txt - 프로그램 실행의 로그 파일 //

main.py - 프로그램 실행의 중추 //

tament_class_map.csv - 모델 결과와 출력 매핑 csv //
